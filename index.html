Taskflow

TaskFlow は、ユーザーが自分のプロジェクトとタスクを直感的に管理できるタスク管理アプリです。

  • 各ユーザーはアカウントを作成し、ログインします。
  • ユーザーはプロジェクトを作成し、そのプロジェクトに紐付けたタスクを追加、編集、削除することができます。
  • タスクは「未着手」、「進行中」、「完了」の状態で管理でき、進捗状況を簡単に確認できます。

GitHub リポジトリ デプロイ先

Aozora

Aozora は，ユーザーが自分の書籍を管理することができるウェブアプリケーションです。

  • ユーザーは青空文庫データベースから書籍を検索し，登録します．
  • 登録した書籍は，自分の書籍一覧に表示され，「未購入」，「未読」，「読書中」，「読了」の状態で管理することができます．
